bridge网络需要的命令

  • 创建一个bridge网络，名称为my_net

    docker network create [bridge名称]

    docker network create my_net

  • 查看docker网络，是否创建成功。

    docker network ls

  • 移除

    docker network rm [bridge名称]

  • 离开自定义网络

    docker network disconnect [bridge名称] [容器名称]
